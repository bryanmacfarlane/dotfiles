# Setup

Setup is idempotent.  Do it once to setup a machine or again if you change files in setup folder

```bash
$ ./setup.sh
```

Remote:
```bash
$ zsh -c "curl -fsSL https://raw.githubusercontent.com/bryanmacfarlane/dotfiles/master/setup.sh"
```
